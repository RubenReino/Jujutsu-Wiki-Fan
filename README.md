# Jujutsu Wiki Fan

Proyecto web inspirado en el universo de Jujutsu, donde se recopila información sobre personajes, habilidades, rangos y más.

##  Ver página

 https://rubenreino.github.io/Jujutsu-Wiki-Fan/

##  Características

* Diseño inspirado en estética japonesa / jujutsu
* Secciones como:

  * Inicio
  * Personajes
  * Galería
* Estilos personalizados con CSS
* Elementos visuales tipo ofuda y pergaminos

## 🛠️ Tecnologías

* HTML
* CSS
* JS

## 📖 Objetivo

Crear una wiki visual atractiva y organizada para practicar desarrollo web y diseño UI inspirado en anime.
